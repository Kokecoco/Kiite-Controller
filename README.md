# Kiite-Controller

## 概要

[Kiite](https://kiite.jp/) の音楽再生を、他のタブやアプリケーションを操作している最中でもグローバルショートカットキーで制御できるようにする、非公式のChrome拡張機能です。

作業中に「ちょっと曲を止めたい」「次の曲に進みたい」と思った時に、わざわざKiiteのタブに戻る必要がなくなります。

**※本拡張機能は有志が作成した非公式のファンツールであり、Kiite公式サイトとは一切関係ありません。**

## 主な機能

-   **再生 / 一時停止**
-   **次の曲へ**
-   **前の曲へ**

これらの操作を、任意のショートカットキーに割り当てることができます。

## 導入方法

#### 1. 拡張機能ファイルのダウンロード

1.  このリポジトリの右側にある [**Releases**](https://github.com/Kokecoco/Kiite-Controller/releases) ページにアクセスします。
2.  最新バージョンの「Assets」セクションから、`Kiite-Global-Controller-vX.X.X.zip` のようなZIPファイルをダウンロードします。
3.  ダウンロードしたZIPファイルを解凍（展開）してください。

#### 2. Chromeへのインストール

1.  Chromeを開き、アドレスバーに `chrome://extensions` と入力して拡張機能管理ページに移動します。
2.  ページの右上にある **「デベロッパーモード」** のスイッチをオンにします。
3.  左上に表示される **「パッケージ化されていない拡張機能を読み込む」** ボタンをクリックします。
4.  ファイル選択ダイアログが開くので、先ほど**解凍したフォルダ**（`manifest.json` が入っているフォルダ）を選択します。
5.  一覧に「Kiite Global Controller」が追加されればインストール完了です。

#### 3. ショートカットキーの設定

1.  Chromeのアドレスバーに `chrome://extensions/shortcuts` と入力してショートカット設定ページに移動します。
2.  「Kiite Global Controller」の項目が表示されているので、各操作の横にある入力欄をクリックし、お好みのキーの組み合わせを設定してください。
    -   他の拡張機能やOSのショートカットと競合しないキーを設定することをお勧めします。（例: `Ctrl + Shift + 9` など）
